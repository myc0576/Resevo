# Target Journal Reviewer Citation Rule

## Decision

SC-NMT and reusable paper-writing workflows should require a target-journal reviewer/editor literature pool before a manuscript submission package is considered complete.

## Rationale

For submission-facing work, missing relevant work by the target journal's regular reviewers, editors, guest editors, or field experts increases review risk. The rule makes this an explicit citation coverage check rather than an informal memory task.

## Boundary

This rule must not become citation gaming.

- Cite only scientifically relevant work.
- Do not guess confidential reviewer identities.
- Do not fabricate reviewer names or approval records.
- Do not add irrelevant citations to appeal to possible reviewers.
- Do not treat citation coverage as evidence for method validity.

## Required Artifact

The submission package should include:

```text
target_journal_reviewer_literature_pool
citation_coverage_table
pending_target_journal marker when the journal is not selected
```
